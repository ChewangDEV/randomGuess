#  Guess The Number Game

A fun and interactive number guessing game built with **HTML, CSS, and JavaScript**.

The computer randomly selects a number between **1 and 100**, and the player must guess the correct number before running out of lives.


##  Features

* Random number generation
* Input validation
* Lives system
* Attempt counter
* Too High / Too Low hints
* Game Over state
* Play Again functionality
* Responsive design
* Modern UI with glassmorphism effect


##  How to Play

1. Click the **Start** button.
2. The computer secretly selects a random number.
3. Enter your guess and click **Submit**.
4. The game will tell you:

   *  Too High
   *  Too Low
   *  Correct Guess
     
5. Each incorrect guess reduces your remaining lives.
6. Guess the number before your lives reach zero.
7. Click **Play Again** to start a new game.
